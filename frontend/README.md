# IMADGEN | AI Marketing Agency

IMADGEN is an AI-native creative studio and marketing operating system. We provide adaptive AI systems and tailored services to help businesses compound their practical intelligence.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (Next.js Modules & Globals) / Tailwind CSS (Reserved)
- **Theming**: [next-themes](https://github.com/pacocoursey/next-themes) for Dark/Light mode support.
- **Components**: Custom React components with a focus on high-end aesthetics and micro-animations.

## 🛠️ Features

- **Hero System**: Dynamic hero sections with blurred background effects.
- **Verticals**: Specialized modules for Creative Studio, Marketing OS, and AI Agents.
- **Interactive UI**:
    - **Cursor Glow**: Mouse-tracking ambient light effects.
    - **Embers Effect**: Animated particle background.
    - **World Clocks**: Real-time interactive analog clocks.
- **Theming**: Seamless dark and light mode transitions.
- **Responsive Design**: Optimized for desktop and mobile devices.

## 📂 Project Structure

- `app/`: Next.js App Router pages and global styles.
- `components/`: Reusable React components.
    - `effects/`: Animation and visual effect components.
    - `ui/`: Base UI primitives (Buttons, Cards, Chips).
- `public/`: Static assets (images, logos, fonts).
- `lib/`: Utility functions and shared constants.

## 🏁 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
